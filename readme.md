# Simple Graphics

## Приложение

Для отрисовки графики использованы средства WINAPI

## Описание кода

- Интерфейс классов фигур и полотна описан в директории **simple_graph_api**
- Реализация интерфейса со средствами WINAPI находится в **winapi_graph**
- Еще одна реализация интерфейса с заглушками вместо кода находится в **template_impl**
- В файле **test.cpp** пример использования **winapi_graph**
